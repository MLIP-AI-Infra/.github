# MLIP-AI-Infra Development Guide

## Project Overview

**MLIP-AI-Infra** is a unified infrastructure repository for accelerating machine learning interatomic potential (MLIP) inference and downstream computational chemistry workflows. The project goal is to maximize inference speed and minimize memory usage while preserving accuracy.

### Components

- **UltraFastUMA** (`UltraFastUMA/`): Mixed-precision inference, torch.compile integration, per-module benchmarking for UMA
- **UltraFastNEB** (`UltraFastNEB/`): Batched NEB image inference, descriptor caching for enzyme systems, optimized NEB engine
- **UltraFastFEP** (`UltraFastFEP/`): Free energy perturbation with lambda-window parallelism (early stage)
